# Todoリストアプリ

## アプリ概要

Vanilla JavaScriptで作成したシンプルなTodoリストアプリ。ブラウザのみで完結し、データはlocalStorageに保存されます。

## 使用技術

- HTML / CSS / JavaScript（Vanilla JS）
- Vite（ビルドツール）
- localStorage（データ永続化）

## 機能一覧

### 基本機能
- Todoの追加（ステータス選択可能）
- **Todoの名称編集**：テキストをクリックして編集可能（Enterで保存、Escでキャンセル）
- Todoの削除
- ステータスの切り替え（未完了 ↔ 処理中 ↔ 完了）
- Todo一覧の表示（ステータスごとに自動ソート）
- ページリロード後もデータを保持

### 追加機能
- **期日設定**：各Todoに期日を設定可能
- **アーカイブ機能**：完了したTodoを一時非表示にして整理
- **ステータスサマリー**：各ステータスの件数を表示
- **通知機能**：操作完了時の視覚的フィードバック

## ステータス

- 未完了（todo）：赤色で表示
- 処理中（doing）：黄色で表示
- 完了（done）：緑色で表示、テキストは打消し線

## 使い方

### Todoの編集
1. Todoのテキスト部分をクリックすると編集モードになります
2. テキストを修正します
3. **Enterキー**を押すか、入力欄の外をクリックすると保存されます
4. **Escキー**を押すと変更をキャンセルできます

### ステータスの変更
- 各Todoの右側にあるドロップダウンから「未完了」「処理中」「完了」を選択できます
- ステータスを変更すると、Todoが自動的に並び替えられます

### 期日の設定
- 各Todoの下部にある日付入力欄から期日を設定できます
- 期日は任意で、設定しなくても問題ありません

### アーカイブ機能
- 「アーカイブ」ボタンをクリックすると、Todoを一時的に非表示にできます
- アーカイブしたTodoは「アーカイブを表示」から確認・復元できます

## 工夫した点

- **モジュール分割**：機能を`todoManager.js`（データ管理）と`todoUI.js`（UI管理）に分けて保守性向上
- **インライン編集**：テキストをクリックするだけで即座に編集可能（Enterキーで保存、Escキーでキャンセル）
- **レスポンシブデザイン**：デスクトップ・モバイル両対応
- **ライト/ダークモード対応**：システム設定に応じて自動切り替え
- **スマートソート**：ステータスと作成日時で自動的に並び替え（完了したTodoは下に移動）
- **ビジュアルフィードバック**：ホバー時のアニメーション、ステータスごとのカラーコード、操作時の通知
- **クリーンなUI**：シンプルで直感的な操作感
- **データバリデーション**：空のテキストや無効な入力を適切に処理

## 主要なデータ構造

```javascript
{
  id: number,              // 一意のID
  text: string,            // Todoの内容
  status: "todo" | "doing" | "done",  // ステータス
  createdAt: string,       // 作成日時（ISO形式）
  archived: boolean,       // アーカイブ状態
  dueDate: string | null   // 期日（YYYY-MM-DD形式、任意）
}
```

## ブラウザ完結

このアプリはサーバーサイド処理を使用せず、ブラウザのlocalStorageのみでデータ管理を行います。
インストール不要でそのまま使用できます。

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

# ビルドしたファイルのプレビュー
npm run preview
```

## 技術的な特徴

- **Vanilla JavaScript**: フレームワーク不要のシンプルな実装
- **ES Modules**: モジュール化による保守性の向上
- **localStorage API**: クライアントサイドでのデータ永続化
- **CSS Variables**: テーマカラーの一元管理
- **Responsive Design**: Flexboxを活用した柔軟なレイアウト
- **Event-driven**: イベントリスナーによるリアクティブなUI更新
