# Todoリストアプリ

## アプリ概要

Vanilla JavaScriptで作成したシンプルなTodoリストアプリ。ブラウザのみで完結し、データはlocalStorageに保存されます。

## 使用技術

- HTML / CSS / JavaScript（Vanilla JS）
- Vite（ビルドツール）
- localStorage（データ永続化）

## 機能一覧

- Todoの追加（ステータス選択可能）
- Todoの削除
- ステータスの切り替え（未完了 ↔ 処理中 ↔ 完了）
- Todo一覧の表示
- ページリロード後もデータを保持

## ステータス

- 未完了（todo）：赤色で表示
- 処理中（doing）：黄色で表示
- 完了（done）：緑色で表示、テキストは打消し線

## 工夫した点

- **モジュール分割**：機能を`todoManager.js`（データ管理）と`todoUI.js`（UI管理）に分けて保守性向上
- **レスポンシブデザイン**：デスクトップ・モバイル両対応
- **ライト/ダークモード対応**：システム設定に応じて自動切り替え
- **ビジュアルフィードバック**：ホバー時のアニメーション、ステータスごとのカラーコード
- **クリーンなUI**：シンプルで直感的な操作感

## 主要なデータ構造

```javascript
{
  id: number,
  text: string,
  status: "todo" | "doing" | "done"
}
```

## ブラウザ完結

このアプリはサーバーサイド処理を使用せず、ブラウザのlocalStorageのみでデータ管理を行います。
インストール不要でそのまま使用できます。
